# Image Processing in C

## Overview
This project implements image processing techniques in C, including grayscale conversion and blur filtering using convolution.

## Features
- Image loading using stb_image
- Grayscale conversion
- Blur filter using convolution
- Image saving using stb_image_write

## Technologies
C, Algorithms, Memory Management

## What I Learned
- Working with pointers and dynamic memory
- Image data processing
- Implementing convolution filters
